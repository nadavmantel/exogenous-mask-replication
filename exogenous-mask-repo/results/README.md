# Results

This folder will contain generated outputs after running the model.

## Expected Files

After running all modes:

| File | Generated by | Description |
|------|--------------|-------------|
| `gb_model_results.csv` | `--mode normal` | Period-by-period results for all scenarios |
| `battery_subsidy_sweep_endogenous.csv` | `--mode battery_sweep` | Battery-only subsidy sweep |
| `solar_battery_subsidy_grid_endogenous.csv` | `--mode solar_battery_sweep` | Solar+battery grid search |
| `figure1_integration_barrier.png/pdf` | `figure1_integration_barrier.py` | Figure 1 |
| `figure2_threshold_dynamics.png/pdf` | `figure2_threshold_dynamics.py` | Figure 2 |

## Regenerating Results

```bash
# Step 1: Run basic comparison
python src/gb_electricity_model.py --mode normal

# Step 2: Run subsidy sweeps (longer)
python src/gb_electricity_model.py --mode battery_sweep
python src/gb_electricity_model.py --mode solar_battery_sweep

# Step 3: Generate figures
python figures/figure1_integration_barrier.py
python figures/figure2_threshold_dynamics.py
```

Note: Subsidy sweeps can take several hours depending on the resolution settings.
